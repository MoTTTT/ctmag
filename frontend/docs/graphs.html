GRAPHS:

MAIN CURVE GRAPH
When the magnetisation curve test is completed, a magnetisation curve is drawn. This curve represents the relationship between the measured current and voltage readings for the CT (Current Transformers) under test, the magnetisation curve kneepoint is displayed in the results panel.

THREE PHASE CURVE
Once a Magnetisation Curve Test have been completed it will be displayed in the maincurve graph (CT Magnetisation Curve). This curve can be assigned to one of three phases and when assigned the main curve graph is redrawn in the three-phase curve graph canvas (3 Phase CT Magnetisation Curve), with the drawn curve set to the colour of the selected phase.
