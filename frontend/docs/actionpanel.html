ACTION BUTTON PANEL

NEW FILE:
This selection clears the application screen, allowing for a new test sequence to be conducted, saved and printed.

SAVE TO FILE:
When selecting this option a FileDialog box appears allowing the user to specify the filename and directory into which to save the serialised test information. The specified filename is displayed in the File Name textfield of the General Settings panel.

IMPORT TEST:
When the Import Test button is selected a a FileDialog box appears allowing the user to import a saved serialised test from a specified directory.

PRINT REPORT:
When selected you can view the report layout. By selecting the OK button in the report dialog box, a printer dialogue box appears. The printed report will contain a header that includes the date, time, and name of the tester/user etceteras. All test settings, results and graphs would be contained in the body of the report. The footer would contain space for an entered signature and date.

EXIT: Exit the application.
