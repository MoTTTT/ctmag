VT (Voltage Transformer) TAB PANEL

VT MAKE:
Choices are displayed and can be selected. If the make is not in the selection then the relevant entry could be entered into the VT combobox text field.

TYPE:
Choices are displayed and can be selected. If the make is not in the selection then the relevant entry could be entered into the Type combobox text field.

CLASS:
Choices are displayed and can be selected. If the make is not in the selection then the relevant entry could be entered into the Class combobox text field.

PHASE:
Make a selection of the choices available e.g. three-phase four-wire.

DESCRIPTION:
Editable text field in which the user can enter descriptive information about the relevant transformer.

SERIAL NO:
Enter the transformer serial number into the text field.

IL:
Editable text field where the user can enter the IL number for the transformer indicating the break down voltage etceteras of the transformer.

RATIO:
The Ratio text field area is divided into two separate entries:

The primary to secondary winding ratio of the voltage transformer would be entered into a dual text field with the primary windings entered as an integer value into the first text field and the secondary windings into the second text field.

 It's important for both values to be entered so that the percentage winding ratio error can be calculated.  It is to be expected that the same turns ratio apply to all three phases in the test set. Although it is possible to change the expected ratio for each of the 3 phases resulting in accurate calculations for each of the phases, only the last entered ratio would be saved for the test set resulting in erroneous results when this test is imported again.

VA RATING:
Enter the Power (Voltage Ampere) rating of the transformer into the text field.
