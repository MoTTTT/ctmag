MENU FUNCTIONS:

FILE

  • New Test   <Ctrl + n> - Instantiate a new test
  • Next Test  <Ctrl + e> - Instantiate a new test while maintaining all the panel settings.
  • Import Test  <Ctrl + i> - Open a FileDialog to allow a test to be imported.
  • Save Test  <Ctrl + s> - Short-cut to save the test to its currently specified filename and directory. If no filename and directory have been specified then a FileDialog is displayed.
  • Save Test As  <Ctrl + a> - Open a FileDialog to allow a file to be saved as a specified filename in a specified directory.
  • Generate Text File <Ctrl + g> -  Open a FileDialog to allow a text file of the magnetisation curve test to be saved. This file will contain the general settings and the CT settings. The voltage and current data points for all three phases as well as the main curve would be included with the measured kneepoints indicated. This text file can be imported into a spreadsheet such as Excel (delimited text file with the limiters being spaces).
  • Print Report  <Ctrl + p> - Open a TestReportDialog to allow a file to be printed if so selected.
  • Links - Links to previously opened and saved tests. Currently not enabled.
  • Exit  <Ctrl + x> - Exit the application, note that tests are not automatically saved on exit.
SETTINGS
  • General Settings - Not enabled.
  • CT(Current Transformer) Settings - Not enabled.
  • VT(Voltage Transformer) Settings - Not enabled.
  • Options:
 (Dialogue Box appears: Selecting OK would implement the current OptionsDialog settings, Cancel would ignore any changes made in the OptionsDialog box )
  1. Select COM port: COM1 or COM2 - Select the serial communications port of the PC to which the    PROMET sensing unit is connected.
  2. Select Magcurve channels: Select the voltage and current input channels for the magnetisation curve test.
  3. Draw upsweep: Include the characteristic curve for the CT when it was initially magnetised.
  4. Draw real time: Draw the characteristic curve for the CT while the test is being conducted.
  5. Real Time Draw Area (10% - 100%): Specify the scaling of the real time curve on the maincanvas.
  6. Cs fullscale current (mA): Specify the fullscale current reading on the Cs channel.
  7. Mag Voltage Multiplication Factor: A multiplication factor to apply to all voltage readings for the magnetisation curve test.
  8. Time Out Threshold (ms) - Specifies the time out period if serial communictions with the PROMET unit is not established on activation of a test procedure.  Note that this value is specified in milliseconds.
  9. Mag Volt Threshold (V at sensor) - Specifies the maximum voltage threshold at which the magnetisation curve test would be aborted.
  10. Mag Current Threshold (mA at sensor) - Specifies the maximum current threshold at which the magnetisation curve test would be aborted.  Note that this value is specified in mA.
  11. Voltage Ratio threshold (V at Vp) - Specifies the maximum measured voltage threshold for the primary winding at which the voltage ratio test would be completed.
  12. Current Ratio threshold (mA at Cp) - Specifies the maximum measured current threshold for the primary winding at which the current ratio test would be completed.  Note that this value is specified in mA.
  TESTS:
  • Assign to RED phase  <Ctrl+ r> - Assign the main curve to the Red phase.
  • Assign to WHITE phase  <Ctrl + w> - Assign the main curve to the White phase.
  • Assign to BLUE phase  <Ctrl + b> - Assign the main curve to the Blue phase.
  • Start Magnetisation Curve Test  <Ctrl + m>
  • Start Current Ratio Test  <Ctrl + c>
  • Start Voltage Ratio Test  <Ctrl + v>
  • Stop Test  <Ctrl + t>
