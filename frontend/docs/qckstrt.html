CT & VT TESTER OPERATING INSTRUCTIONS

TEST SYSTEM COMPONENTS

  • Tester Unit: Promet Metering Hardware
  • Tester to PC serial cable
  • Transformers to be tested
    • Current Transformers
    • Voltage Transformers
  • PC/Laptop with "Promet CTMag Application" installed
PREPARE TEST EQUIPMENT
  • Connect tester unit to PC and power up PC
  • Complete "Options" settings panel
  • Power up tester unit
  • Start CTMag Application
  • Complete General Settings Panel
PREPARE THE CURENT TRANSFORMER (CT) TEST
  • Complete CT Settings Panel
  • Connect CT, Variac and Power Supply
  • Apply power to the system
EXECUTE CT TEST
  • Press "Start Magcurve Test" button
  • Increase supply voltage slowly (e.g. 10V/sec) until kneepoint is reached
  • Decrease supply voltage slowly until variac control knob at 0V setting
  • Wait for the input readings to stabalise, this is indicated by an end of test condition
    • audible beep
    • status textfield below the menu-bar indicates an end of test
    • main graph rescales
    • Test Button Panel changes to reflect an end of test mode
